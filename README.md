# MoneyMate

MoneyMate adalah aplikasi Android sederhana untuk mencatat transaksi keuangan harian, baik pemasukan maupun pengeluaran.

## Deskripsi Aplikasi
Aplikasi ini dibuat untuk memenuhi tugas UAS Pemrograman Mobile dengan menerapkan minimal 4 interface, yaitu:
- form pendataan,
- tampilan data dari form,
- daftar data menggunakan RecyclerView,
- detail data dari RecyclerView.

## Fitur
- Tambah transaksi
- Preview data transaksi
- Lihat daftar transaksi
- Lihat detail transaksi
- Hapus transaksi
- Menampilkan total pemasukan, pengeluaran, dan saldo

## Struktur Halaman
- **MainActivity** → halaman utama
- **InputActivity** → form input transaksi
- **PreviewActivity** → preview data
- **ListActivity** → daftar transaksi
- **DetailActivity** → detail transaksi

## Teknologi yang Digunakan
- Kotlin
- Android XML Layout
- RecyclerView
- Intent
- ViewModel
- Repository sederhana

## Cara Menjalankan
1. Buka project di Android Studio
2. Tunggu proses Gradle selesai
3. Jalankan aplikasi pada emulator atau device Android
